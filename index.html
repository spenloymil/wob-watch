How to add episodes (super simple)

  1. Upload your video to a public place (recommended: YouTube as *Unlisted* for big files).
  2. Open content.json in your GitHub repo and click ✏️ (Edit file).
  3. Find the season you want and paste a new episode object (example on the right).
  4. Commit changes. Refresh the site after ~30–60s.

For small MP4s (<100MB) you may store them in the repo at /videos/ and reference the URL.

Episode JSON (paste inside a season's episodes array): 
,
{
  "id":"ep_my_first",
  "title":"Episode 1 — Pilot",
  "desc":"Short description",
  "videoUrl":"https://youtu.be/ABCDEFG" 
}
Made for WOB WATCH+. To publish: put this `index.html` (and `content.json`) in your repo root and enable GitHub Pages.
